# Phase 1 Core Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the minimal UI-independent mocode core for YAML path lookup, schema hover/completion, semantic indexing, and basic Mihomo diagnostics.

**Architecture:** Keep text storage in `mocode-text`, YAML CST and path lookup in `mocode-yaml`, static Mihomo docs in `mocode-mihomo-schema`, semantic extraction/validation in `mocode-mihomo-lint`, and orchestration in `mocode-core`. The app crate remains an adapter only and should not receive Mihomo logic.

**Tech Stack:** Rust 2024, `ropey`, `tree-sitter`, `tree-sitter-yaml`, focused unit tests, fixture YAML files.

---
